# ML_Project25
Classification + Spam Email Detection using Machine Learning

Dependencies: PyTorch, NumPy, Scikit Learn

To Install:

pip install torch
pip install numpy
pip install scikit-learn

To Run:

Ensure python + dependencies are downloaded
Run both .py files, and watch terminal.
Result files will be produced (ClassificationTestResultSN for SVM, ClassificationTestResultN for NN)
